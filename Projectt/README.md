# Birlikte İzle

Sevdiklerinizle birlikte online video izleme platformu.

## Özellikler
- Senkronize video izleme
- Sesli/görüntülü sohbet
- Ekran paylaşımı
- Playlist yönetimi
- Emoji reaksiyonları

## Demo
https://your-app.railway.app

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express
- Real-time: Socket.io
- WebRTC: Peer-to-peer communication
- Deployment: Railway
